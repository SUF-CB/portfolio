My Projects

During my free time on weekends I take up sometime to create programs and websites as a hobby. By doing this I am becoming more stronger and more knowledgeable so I can fullfill my highest and beyond potential in the Computing industry.

Below are the projects which I have done so far:


Project 1-Multplication Quiz

This is a Command Line Interface Multiplication Quiz aimed at children to test them on their times-tables. Each child gets asked Ten random questions based on any of the tables between 1-12. At the end of the quiz your child will recieve a score out of ten based on their correct answers.



Project 2-Hotel Booking Application

This is a Windows Form Application where customers can book a hotel along with some additional extras for themselves and their families.




Project 3-Web Servers Website

This website talks briefly about the security issues and the possible solutions to prevent the threats to Web servers. As well as this,there is also a newsletter which users can subscribe to so they can continuously stay alert on the threats causing issues and as well as solutions to effectively prevent the issues.



Project 4-Cinema Booking System

This a Command-Line Interface Cinema Booking System where customers can pick a film and book different types of seats for themselves and their families.



Project 5-Prime Number Generator

This program is for maths students who are preparing for their maths GCSE'S. The aim of this program is to check if the students know everything about prime numbers.



Project 6-Pizza Delivery Website

This is a Pizza Delivery site where customers can order pizzas with additional extras in order to kill their hunger!



Project 7-Programming Website

Project 8- Calculator Program
